Metode Instalasi

1

Auto Installer (Recommended)

Cocok untuk STB Armbian, Linux Server, dan Termux. Script ini otomatis menginstall Node.js, Git, PM2, dan mengatur Auto-Start saat reboot.

Terminal (Bash)
curl -sL https://raw.githubusercontent.com/z1dan820/hazivault/main/install.sh | sudo bash
Untuk Termux: Hapus kata "sudo" pada perintah di atas.
2

Docker Container

Metode paling bersih dan terisolasi. Tidak perlu install dependency manual. Cukup jalankan container.

Docker CLI
docker run -d \ --name hazivault \ --restart unless-stopped \ -p 3000:3000 \ -v $(pwd)/data:/app/data \ ghcr.io/z1dan820/hazivault:latest
3

CasaOS Custom Install

Instalasi visual melalui dashboard CasaOS. Pilih "Install a Custom App" dan isi data berikut:
